1.Resumen

El modulo Usuario permite el manejo de las cuentas de usuarios de la Intranet, además de la administración de áreas de trabajo, perfiles de usuario y de los anexos telefónicos.

Un área de trabajo representa las divisiones en la organización de la UOCT. De esta forma se pueden mencionar, por ejemplo, el área de Inspección técnica, el área de Sala de Control, el área de Diseño, etc. La intranet está diseñada de manera flexible, permitiendo que se agregen o eliminen áreas de trabajo de manera simple, solamente con la restricción de mantener un área de INGENIERÍA, debido a que existen funcionalidades, como por ejemplo hacerse cargo del análisis de un EIV, que corresponden solamente a usuarios que pertenecen a ésta área.

Un perfil de usuario está compuesto por un conjunto de permisos que acotan las funcionalidades de los usuarios de la intranet. Cada usuario debe estar asociado a un perfil de usuario.

Una cuenta de usuario es un registro de una persona de la institución. Esta cuenta permite sesiones personalizadas de trabajo en la intranet, por medio del uso de nombre de usuario y password.

Cada usuario puede editar sus datos personales, como nombres, dirección, anexo telefonico dentro de la institucion, números de telefono de contacto, fecha de cumpleaños, foto, curriculum, etc. Además se puede registrar los informes de actividades, con lo cual, solamente los usuarios autorizados pueden descargarlos.

Además, los datos de los usuarios quedan disponibles para ser vistos por otros usuarios de la intranet.

Asi como cada usuario puede registrar su anexo telefónico, se pueden mantener otro conjunto de anexos telefónicos que no pertenecen necesariamente a un usuario de la intranet, como por ejemplo, portería.

Cada usuario de la intranet puede consultar un cuadro de anexos telefónicos que incluye, tanto los anexos de los usuarios de la intranet, como los anexos que no pertenecen a usuarios.

2. Áreas de Trabajo

Administrar áreas de trabajo (Restringido a personal autorizado):

Para agregar o elminar un área de trabajo:

  1. Expandir menu Usuario en la izquierda de la pantalla.
  2. Seleccionar "admin. áreas de trabajo".
  3. Se mostrará una página con un formulario para ingresar una nueva área y una lista con todas las áreas ya ingresadas en la intranet.
  4. Para agregar una nueva área ingrese el nombre de la nueva área y presione el boton "Agregar".
  5. Para eliminar una área existente, presione el botón eliminar al lado del área que desea eliminar( Se eliminará el área solamente si no existen usuarios, ni elementos del inventario asociados a ella.

3. Perfil

Agregar Perfil (Restringido a personal autorizado)

  1. Expandir menu Usuario desde la izquierda de la pantalla.
  2. Seleccionar "Agregar Perfil".
  3. Se mostrará una pantalla con el formulario para el ingreso del perfil donde se indican las operaciones disponibles.
  4. Indique el nombre del perfil y selecciones las atribuciones que desea para ese perfil
  5. Persione el boton "ingresar" al final del formulario.

    Editar Perfil (Restringido a personal autorizado)

  1. Expandir el menu de Usuario desde la izquierda de la pantalla
  2. Seleccione " Editar perfil".
  3. Se mostrará una lista de los perfiles disponibles con las opciones "Ver Detalla" y "Editar" para cada uno de ellos.
  4. Si presiona Editar se mostrara una pantalla con las operaciones de la intranet y una casilla para habilitar o deshabilitar tal operación en el perfil seleccionado.
  5. Luego de realizar las ediciones deseadas presione el botón guardar al final del formulario.
  6. Si desea eliminar el perfil, al final del formulario se encuentra un boton que permite esta operación.Sólo se eliminará el perfil en caso que no hayan usuarios asociados a tal perfil.

3.Usuario

Agregar Usuario (Restringido a personal autorizado)

  1. Seleccione la opción "agregar usuario" desde el menu a la izquierda de la pantalla.
  2. Se mostrará un formulario para ingresar los datos del nuevo usuario, entre los que se incluye el username y password que usará para ingresar a la Intranet, el perfil del usuario y el área de trabajo.

Administrar Usuario (Restringido a personal autorizado)

  1. Seleccionar la opción "administrar usuarios" desde el menu a la izquierda de la pantalla.
  2. Se mostrará una lista de los usuarios de la intranet con la opción "editar" al lado de cada uno de ellos
  3. Al seleccionar editar algun usuario se mostrará una pantalla con los datos editables de cada cuenta de usuario, además de la opción "desactivar" cuando el usuario esté activo, o "activar" si el usuario está desactivado.
  4. Luego de editar los datos del usuario, presionar el botón "Guardar los cambios".

Ver Datos de los Usuarios

  1. Seleccionar la opción "Ver datos de los usuarios" en el menú a la izquierda de la pantalla.
  2. Se mostrara una lista con todos los usuarios de la intranet.
  3. Al cliquear un nombre de usuario se mostrarán los datos del usuario seleccionado.

Cumpleaños

  1. Seleccionar "cumpleaños" en el menú a la izquierda de la pantalla.
  2. Se mostrará una lista con todos los usuarios de la intranet y su respectiva fecha de cumpleaños, en caso que esté registrada.

Editar Datos Personales

  1. Seleccionar "editar datos personales" en el menú a la izquierda de la pantalla.
  2. Se mostrará una pantalla con todos los datos personales editables por el usuario. Además, dentro de la misma pantalla aparecerán otras opciones para editar datos como la contraseña de la cuenta de intranet,actualizar el curriculum, ingresar informe de actividades y cambiar la fotografía personal.
  3. Luego de editar los datos, presionar el boton "guardar los cambios".
  4. Para cambiar el curriculum debe seleccionar la opción "cambiar curriculum" en la pantalla de edición de datos personales y posteriormente ingresar el nuevo curriculum actualizado, con lo que se eliminará de la intranet el curriculum antiguo.
  5. Para ingresar o eliminar un informe, seleccione la opción "Informe de actividades desde la pantalla de edición de datos personales", se mostrará una lista con los informes registrados con la opción "eliminar" para cada uno de ellos, además de el formulario para el ingreso de nuevos informes.

5.Anexos Telefónicos

Cuadro de Anexos telefónicos

  1. Seleccionar la opcion "anexos telefónicos" desde el menu a la izquierda de la pantalla.
  2. Se mostrará una lista con todos los usuarios de la intranet y su correspondiente anexo telefónico en caso que esté registrado. Además, en esta lista se muestran los anexos telefónicos que no pertenecen a usuarios en caso que existan.

Administrar Anexos telefónicos (Restringido a personal autorizado)

  1. Seleccionar la opcion "administrar anexos telefónicos" desde el menu a la izquierda de la pantalla.
  2. Se mostrará una lista de los anexos que no pertenecen a usuarios, con la opción "Editar" para cada uno de ellos. Además se muestra un formulario para ingresar nombre y número de un anexo nuevo.
  3. Al seleccionar editar un anexo, se podrá editar el número del anexo o eliminarlo.
